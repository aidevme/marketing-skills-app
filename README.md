# Marketing Skills App

## Overview

A web application for managing and developing marketing skills.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

### Building for Production

```bash
npm run build
```
